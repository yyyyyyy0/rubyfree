# rubyfree

マウスを画面上の漢字語にホバーして静止させると、透明オーバーレイにふりがな（ルビ）を自動表示する macOS 常駐ユーティリティです。日本語学習の補助を目的としています。

> Windows 専用の「ホバーでふりがな」系ツールの macOS ネイティブ移植（個人開発・OSS）。

## 特徴 / 設計原則

- **完全ローカル・非通信**: 取得したテキストや画面内容を **一切外部送信しません**。テレメトリ・解析・自動アップデート通信もありません。著作権が発生するテキストを扱う前提のため、非通信を中核要件としています。
- **非永続**: 取得テキスト・OCR 画像・生成した読みを **ディスクやログに残しません**（保存するのは設定値のみ）。
- **最小権限**: アクセシビリティ権限のみで動作します。OCR フォールバックは **オプトイン** で、画面収録権限は OCR が実際に必要になった時にのみ要求します。
- **プライバシー保護**: パスワード入力欄（セキュアフィールド）や除外指定アプリ上では取得を行いません。

## 動作環境

- macOS 26 以降 / Apple Silicon
- Swift 6.2 以降（ビルド時）

## インストール（推奨: ソースからビルド）

本アプリはアクセシビリティと画面収録という強力な権限を持つため、**配布元の真正性が重要**です。最も安全な導入方法は自分でソースからビルドすることです（ダウンロードした実行ファイルに付く隔離属性も付きません）。

```sh
git clone https://github.com/yyyyyyy0/rubyfree.git
cd rubyfree
./Scripts/build-app.sh        # rubyfree.app を生成
./Scripts/run-dev.sh          # 固定パスへ配置して起動
```

初回起動時にアクセシビリティ権限を求められます。「システム設定 → プライバシーとセキュリティ → アクセシビリティ」で許可してください。

> バイナリ配布（GitHub Releases）を利用する場合は、併記された SHA-256 を照合し、初回は **右クリック → 開く** で起動してください。`xattr` による隔離属性の一括解除は推奨しません。

## 既知の制約

- **AX 非対応アプリ**: Chrome / 一部 Electron アプリ / 画像・PDF 内の文字はアクセシビリティ API で取得できません。OCR フォールバック（オプトイン）で対応します。
- **読みの精度**: 標準の言語処理を用いるため、固有名詞・人名・熟字訓など文脈依存の語は誤読することがあります。**不確実な読みは淡色で表示** して区別します（補助的なルビとしてご利用ください）。
- **縦書き / 一部の全画面アプリ**: v1 では非対応です。

## 開発

### コーディング規約（プライバシー / セキュリティ上の必須事項）

- **ユーザーデータ非永続**: AX/OCR で取得したテキスト・画像・読みを、ファイル・キャッシュ・`UserDefaults` に書き込まないこと。履歴も持たない。保持はパイプライン通過中の最短時間に限る。
- **ログにユーザーデータを出さない**: `os_log` に生テキストを渡さない（`%{public}` 禁止、ユーザーデータを含む `print` 禁止）。診断ログは値型のメタデータのみ。
- **デバッグ出力は `#if DEBUG` で囲む**: リリースビルドに残さない。
- **ネットワーク API を使わない**: `URLSession` / `Network` / ソケット等を import・呼び出ししない（CI と `Scripts/audit-binary.sh` で検査）。
- **メモリのゼロクリアは約束しない**: Swift の値型では保証できないため、保持時間の最短化で代替する。

### レイヤ構成

- `RubyfreeCore` — 純ドメインロジック（AppKit 非依存、カバレッジ 80% 対象）
- `RubyfreeSystem` — OS 境界ラッパ（AX / ScreenCaptureKit / Vision / 入力監視）
- `rubyfree` — AppKit メニューバー常駐エグゼキュータブル（composition root）

### コマンド

```sh
swift build
swift test --enable-code-coverage     # Core のユニットテスト
./Scripts/pre-push.sh                 # ローカル全検証（build + test + .app ビルド + バイナリ監査）
./Scripts/audit-binary.sh rubyfree.app  # 非通信のバイナリ監査
```

詳細な計画は開発者向けのプラン（リポジトリ外）に基づきます。
