# プライバシー方針

rubyfree は **完全ローカル・非通信** で動作する macOS ユーティリティです。著作権が発生しうるテキストを扱う前提のもと、ユーザーの画面内容・入力内容を保護することを最優先に設計しています。

## 収集・送信しないもの

- rubyfree は **いかなるデータも外部に送信しません**。ネットワーク通信を行うコードを含みません（`URLSession` / `Network` 等を使用せず、CI と `Scripts/audit-binary.sh` のバイナリ監査で検査しています）。
- テレメトリ・利用解析・クラッシュレポート送信 SDK・自動アップデート通信を **一切含みません**。

## 取得データの扱い

- アクセシビリティ API または（オプトイン時の）画面 OCR で取得したテキスト・画像は、**メモリ内でルビ生成に使った後すぐに破棄** します。
- 取得テキスト・OCR 画像・生成した読みを **ディスク・ログ・キャッシュに保存しません**。`UserDefaults` に保存するのは設定値（デバウンス時間・OCR の有効/無効など）のみです。
- 履歴機能を持ちません。

## 権限について

- **アクセシビリティ（必須）**: フォーカス位置周辺のテキストを読み取るために使用します。
- **画面収録（任意・OCR 有効時のみ）**: アクセシビリティで取得できないアプリ（ブラウザ・画像内文字など）向けのフォールバックです。OCR を有効にした場合にのみ、実際に必要になった時点で要求します。
- **マウス位置の監視**: カーソルの静止を検知してルビ表示の起点にします。座標はメモリ内処理のみで保存・送信しません。メニューバーから OFF にすると監視を完全に停止します。

## 取得を抑止する場面

- パスワード入力欄（セキュアテキストフィールド）の上では、テキスト取得も OCR も行いません。
- 既知の機微なアプリ（パスワードマネージャ等）を除外対象として扱います。

## 推奨設定

- 強権限を持つユーティリティ一般の注意として、macOS の「分析と改善（診断データの Apple との共有）」を **オフ** にしておくことを推奨します（万一のクラッシュ時に診断レポートが外部へ渡る経路を断つため）。

## 悪用の禁止

本ソフトウェアは学習補助を目的としています。非通信はその中核要件です。この非通信ガードを除去して画面・入力内容を外部送信するような派生物の配布は、本プロジェクトの趣旨に反します。
